The experiments conducted in the Human Activity Recognition Using Smartphone dataset are summaried by subject and activity.

For each summarized record it is provided:
======================================

 
- An identifier of the subject who carried out the experiment.
- Its activity label.
- average Triaxial acceleration from the accelerometer (total acceleration) and the estimated body acceleration.
- average Triaxial Angular velocity from the gyroscope. 
- average A 561-feature vector with time and frequency domain variables. 


The dataset includes the following files:
=========================================

- 'README.txt'

- 'summary_info.txt': Shows information about the variables used on the feature vector.

- 'Summary.txt': summarization of dataset Human Activity Recognition Using Smartphone
